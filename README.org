* AURIX TC275 hack笔记
** 概述
TC275是我现在工作中经常接触的一个MCU，但是我对这个MCU的熟悉度不是很够。由于现在工作性质的变化，工作之中我基本上抽不出来完整的时间做相应的技术学习了。这给我的工作带来了一定的不顺，但是也给我生活中的hack又开了一个新的方向。官方的开发板已经拿到并且焊接完毕，接下来可以进入hack了，希望学习顺利，happy hacking!
